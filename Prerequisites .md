# Pre-requisites

## Description

The Credit Card Approval Prediction project is developed using Python and various machine learning libraries for data analysis, model training, visualization, and deployment. Before running the project, install the following software and packages.

## Software Requirements

### 1. Python
- Version: Python 3.10 or above
- Used for implementing the machine learning model.

### 2. Visual Studio Code (VS Code)
- IDE used for writing and running Python code.
- Install the Python extension for VS Code.

### 3. Git
- Used for version control and pushing the project to GitHub.

## Python Libraries

### NumPy
Provides support for numerical computations and arrays.

### Pandas
Used for loading, cleaning, and analyzing datasets.

### Matplotlib
Used to create graphs and visualizations.

### Seaborn
Provides advanced statistical visualizations.

### Scikit-learn
Used for data preprocessing, model training, prediction, and evaluation.

### Joblib
Used to save and load the trained machine learning model.

### Flask (or Streamlit)
Used to build a simple web application for predicting credit card approval.

## Installation

Install all required libraries using:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn flask joblib
```

## Hardware Requirements

- Processor: Intel Core i3 or above
- RAM: 4 GB minimum (8 GB recommended)
- Storage: 1 GB free disk space

## Operating System

- Windows 10/11
- Linux
- macOS
